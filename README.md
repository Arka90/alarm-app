# How to run locally

- Step 1 : Download the Code
- Step 2 : Extract the zip file
- Step 3 :  Double Click on the index.hyml file


# How to edit
- Step 1 : Download the Code
- Step 2 : Extract the zip file
- Step 3 : Select the file you want to edit
- Step 4 : Right click
- Step 5 : Choose open with Notepad or any Code editor you have
